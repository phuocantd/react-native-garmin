# react-native-garmin

A sample connect Garmin oauth in react native

## Install
```
  clone source
  cd react-native-garmin
  yarn (npm install)
```

Create file `.env` at the root
```.env
GARMIN_CONSUMER_KEY=
GARMIN_CONSUMER_SECRET=
```

## Android
```
  yarn android (npm run android)
```

## iOS
```
  cd ios && pod install
  yarn ios (npm run ios)
```
